## Usage

Use `<sa-avatar>` to display a single avatar. Display a group of avatars using `<sa-avatar-group>`.

```razor
<!-- Display a single avatar -->
<sa-avatar src="..." />

<!-- Display a group of avatars -->
<sa-avatar-group>
    <sa-avatar src="..." />
    <sa-avatar src="..." />
</sa-avatar-group>
```
