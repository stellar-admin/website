The Empty Tag Helper allows you to display an empty state.

<Demo src="/demo/tag-helpers/empty-intro.html">
  <include>./_include/empty-intro.mdx</include>
</Demo>

## Usage

```razor
<sa-empty>
    <sa-empty-header>
        <sa-empty-media>...</sa-empty-media>
        <sa-empty-title>...</sa-empty-title>
        <sa-empty-description>...</sa-empty-description>
    </sa-empty-header>
    <sa-empty-content>
        ...
    </sa-empty-content>
</sa-empty>
```
