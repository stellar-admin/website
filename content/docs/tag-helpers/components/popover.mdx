## Usage

```razor
<dui-popover>
    <dui-popover-trigger>
        <!-- add the element that triggers the popover here -->
    </dui-popover-trigger>
    <dui-popover-content>
        <!-- add the popover content here -->
    </dui-popover-content>
</dui-popover>
```

## Examples

### Button as trigger

<ProNotice componentName="as-child" className="mb-6" />

The `dui-popover-trigger` is rendered as a button. This creates issues when you want to add a button element inside the trigger as you cannot use nested `button` elements.

To work around this limitation, make sure you add the [`as-child` Tag Helper attribute](/docs/tag-helpers/components/as-child) to `dui-popover-trigger`. This will merge the attributes and behavior of `dui-popover-trigger` with its child content and render a single `button` element.

</Callout>
