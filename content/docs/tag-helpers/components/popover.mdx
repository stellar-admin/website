## Usage

```razor
<sa-popover>
    <sa-popover-trigger>
        <!-- the element that triggers the popover -->
    </sa-popover-trigger>
    <sa-popover-content>
        <!-- the popover content -->
    </sa-popover-content>
</sa-popover>
```

## Examples

### Button as trigger

The `sa-popover-trigger` is rendered as a button. This creates issues when you want to add a button element inside the trigger as you cannot use nested `button` elements.

To work around this limitation, make sure you add the [`as-child` Tag Helper attribute](/docs/tag-helpers/components/as-child) to `sa-popover-trigger`. This will merge the attributes and behavior of `sa-popover-trigger` with its child content and render a single `button` element.
