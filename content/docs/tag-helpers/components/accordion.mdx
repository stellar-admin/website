## Usage

Use `<sa-accordion>` to wrap a collection of accordion items. Individual items are created using the `<sa-accordion-item>` element containing `<sa-accordion-item-trigger>` and `<sa-accordion-item-content>` which contains the title and content respectively.

```razor
<sa-accordion>
    <sa-accordion-item>
        <sa-accordion-item-trigger>
            ...
        </sa-accordion-item-trigger>
        <sa-accordion-item-content>
            ...
        </sa-accordion-item-content>
    </sa-accordion-item>
    <sa-accordion-item>
        ...
    </sa-accordion-item>
</sa-accordion>
```

## Examples

### Single

Allow only a single item to be open at a time using the `type` attribute. When allowing only a single open item, the `collapsible` attribute indicates whether the item can be closed or not.
