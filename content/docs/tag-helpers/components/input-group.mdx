## Usage

```razor
<sa-input-group>
    <!-- 1. Add inputs -->
    <sa-input-group-input/>
    <!-- or -->
    <sa-input-group-textarea/>
    <!-- 2. Add addons -->
    <sa-input-group-addon align="...">
        <!-- you can add text inside the addon... -->
        <sa-input-group-text>...</sa-input-group-text>
        <!-- ...or buttons -->
        <sa-input-group-button/>
    </sa-input-group-addon>
</sa-input-group>
```
