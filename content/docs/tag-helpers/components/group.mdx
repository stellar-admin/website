Arrange elements and components within a horizontal flex container.

## Usage

```razor
<dui-group>...</dui-group>
```

## Examples

### Gap

Control the horizontal gap between items with the `gap` attribute. You can specify a custom gap by adding a `class` from [Tailwind's `gap-` utility](https://tailwindcss.com/docs/gap).

</Demo>
