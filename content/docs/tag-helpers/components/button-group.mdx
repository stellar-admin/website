A button group allows you to group related buttons into sections.

<Demo src="/demo/tag-helpers/button-group-intro.html">
  <include>./_include/button-group-intro.mdx</include>
</Demo>

## Usage

```razor
<dui-button-group>
    <dui-button>...</dui-button>
    <dui-button>...</dui-button>
</dui-button-group>
```
