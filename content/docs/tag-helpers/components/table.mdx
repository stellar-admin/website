## Usage

```razor
<sa-table>
    <sa-table-caption>...</sa-table-caption>
    <sa-table-header>
        <sa-table-row>
            <sa-table-head>...</sa-table-head>
        </sa-table-row>
    </sa-table-header>
    <sa-table-body>
        <sa-table-row>
            <sa-table-cell>...</sa-table-cell>
        </sa-table-row>
    </sa-table-body>
    <sa-table-footer>
        <sa-table-row>
            <sa-table-cell>...</sa-table-cell>
        </sa-table-row>
    </sa-table-footer>
</sa-table>
```
