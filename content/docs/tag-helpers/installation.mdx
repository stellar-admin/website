## Manual Installation

The following installation instructions assume you are adding StellarAdmin to an existing ASP.NET MVC/Razor Pages project.

<Steps>
    <Step>
        ### Install package

        Install the StellarAdmin NuGet package.

        <Tabs items={['.NET CLI', 'Package Manager']}>
            <Tab>
                ```text
                dotnet add package StellarAdmin.TagHelpers --prerelease
                ```
            </Tab>
            <Tab>
                ```text
                Install-Package StellarAdmin.TagHelpers -IncludePrerelease
                ```
            </Tab>
        </Tabs>
    </Step>
    <Step>
        ### Register services

        Update your `Program.cs` (or `Startup.cs`) to register the StellarAdmin services.

        ```cs
        using StellarAdmin;

        builder.Services.AddStellarAdmin();
        ```
    </Step>
    <Step>
        ### Update imports

        Update your `_ViewImports.cshtml` to register the StellarAdmin Tag Helpers.

        ```razor
        @addTagHelper *, StellarAdmin.TagHelpers
        ```
    </Step>
    <Step>
        ### Include stylesheet

        In your `_Layout.cshtml` file (or whichever layout file you use), include the StellarAdmin stylesheet in the `head` section.

        ```razor
        <head>
            <!-- ... -->
            <link rel="stylesheet" href="/_content/StellarAdmin.TagHelpers/stellar-admin.css"/>
        </head>
        ```
    </Step>
</Steps>
