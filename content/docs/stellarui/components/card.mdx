## Usage

```razor
<sa-card>
    <sa-card-header>
        <sa-card-title>...</sa-card-title>
        <sa-card-description>...</sa-card-description>
    </sa-card-header>
    <sa-card-content>
        ...
    </sa-card-content>
    <sa-card-footer>...</sa-card-footer>
</sa-card>
```
