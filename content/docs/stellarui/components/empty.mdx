The Empty Tag Helper allows you to display an empty state.

<Demo src="/demo/tag-helpers/empty-intro.html">
  <include>./_include/empty-intro.mdx</include>
</Demo>

## Usage

```razor
<sui-empty>
    <sui-empty-header>
        <sui-empty-media>...</sui-empty-media>
        <sui-empty-title>...</sui-empty-title>
        <sui-empty-description>...</sui-empty-description>
    </sui-empty-header>
    <sui-empty-content>
        ...
    </sui-empty-content>
</sui-empty>
```
