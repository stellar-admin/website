Arrange elements and components within a vertical flex container.

## Usage

```razor
<sui-stack>...</sui-stack>
```

## Examples

### Gap

Control the vertical gap between items with the `gap` attribute. You can specify a custom gap by adding a `class` from [Tailwind's `gap-` utility](https://tailwindcss.com/docs/gap).

</Demo>
