### Implicit fields

Whenever you use any if the various field input Tag Helpers (such as `<sui-input>`, `<sui-textarea>`, `<sui-select>` etc.) along with data binding, StellarUI will implicitly create a field with a label, description, and placeholder for you. You can see this demonstrated in the example below which implements the same form as above, but using data binding.

<Demo src="/demo/tag-helpers/field-implicit.html">
<Tabs items={["Razor", "Model"]}>
    <Tab value="Razor">
      <include>./_include/field-implicit.mdx</include>
    </Tab>
    <Tab value="Model">
      ```cs
      public class ImplicitModel
      {
          [Display(
              Name = "Card Number",
              Description = "Enter your 16-digit card number",
              Prompt = "1234 5678 9012 3456"
          )]
          public string? CardNumber { get; set; }

          [Display(Name = "Comments", Prompt = "Add any additional comments")]
          public string? Comments { get; set; }

          [Display(Name = "CVV")]
          public string? Cvv { get; set; }

          [Display(Name = "Same as shipping address")]
          public bool IsBillingAddressSame { get; set; }

          public string? Month { get; set; }

          [Display(Name = "Name on Card", Prompt = "Ibn Battuta")]
          public string? Name { get; set; }

          public string? Year { get; set; }
      }
      ```
    </Tab>

  </Tabs>

</Demo>

<Demo src="/demo/tag-helpers/field-input.html">

### Field Group

Stack `<sui-field>` components with `<sui-field-group>`. Add `<sui-field-separator>` to divide them. When grouping checkboxes or radio buttons, you can add `data-slot="checkbox-group"` or `data-slot="radio-group"` to tighten up spacing between the individual inputs.

<Demo src="/demo/tag-helpers/field-field-group.html">
  <Tabs items={["Using Explicit Fields", "Using Implicit Fields", "Model"]}>
    <Tab value="Using Explicit Fields">
        <include>./_include/field-field-group.mdx</include>
    </Tab>
    <Tab value="Using Implicit Fields">
        <include>./_include/field-field-group-implicit.mdx</include>
    </Tab>
    <Tab value="Model">
        ```cs
        public class FieldGroupImplicitModel
        {
            [Display(Name = "Push notifications")]
            public bool EnableResponsePushNotifications { get; set; }

            [Display(Name = "Email notifications")]
            public bool EnableTaskEmailNotifications { get; set; }

            [Display(Name = "Push notifications")]
            public bool EnableTaskPushNotifications { get; set; }
        }
        ```
    </Tab>

  </Tabs>
</Demo>
