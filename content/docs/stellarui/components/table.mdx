## Usage

```razor
<sui-table>
    <sui-table-caption>...</sui-table-caption>
    <sui-table-header>
        <sui-table-row>
            <sui-table-head>...</sui-table-head>
        </sui-table-row>
    </sui-table-header>
    <sui-table-body>
        <sui-table-row>
            <sui-table-cell>...</sui-table-cell>
        </sui-table-row>
    </sui-table-body>
    <sui-table-footer>
        <sui-table-row>
            <sui-table-cell>...</sui-table-cell>
        </sui-table-row>
    </sui-table-footer>
</sui-table>
```
