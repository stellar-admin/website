---
title: Button
---

Displays a button component that can be used to trigger actions like submitting forms or triggering JavaScript functions.

## Usage

```razor
<sui-button>Proceed</button>
```

## Examples

### Default

If you do not specify any attributes, the button will be rendered using the default variant (`ButtonVariant.Default`) and size (`ButtonSize.Default`).

### Size

Specify the size using the `size` attribute.

### Variant

Specify the variant using the `variant` attribute.

### Icons

### Specify additional attributes

`sui-button` is a wrapper around the HTML Button (`<button>`) element. You can use any of the [Button element
attributes](https://developer.mozilla.org/en-US/docs/Web/HTML/Reference/Elements/button#attributes).
